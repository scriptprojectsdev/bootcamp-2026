# Ejercicios del Coding Bootcamp 2026

Los ejercicios se entregarán a través de [GitHub.com](https://github.com) siguiendo los siguientes pasos. 
1. Crea una cuenta en [https://github.com/signup](https://github.com/signup)
2. Crea un repo con el nombre que desees (i.e.: bootcamp-2026)
3. Los ejercicios estarán en [https://github.com/scriptprojectsdev/bootcamp-2026](https://github.com/scriptprojectsdev/bootcamp-2026)
4. Cuando se emita un ejercicio en la clase lo subirás al repo que creaste
   - Para subir un archivo (lo más recomendado) sigue estos pasos:
     - En la parte superior dale "Add file" ó "Añadir archivo"
     - Luego dale a "Upload file" ó "Subir Archivo", asegúrate que tu archivo tenga el mismo nombre que ejercicio
   - Para crear el archivo directamente en GitHub sigue estos pasos:
     - En la parte superior dale "Add file" ó "Añadir archivo"
     - Luego dale a "Create new file" ó "Crear nuevo archivo"
     - Pon el nombre del archivo en la parte superior donde dice "Name your file..." ó "Nombra tu archivo..."
     - Pega el contenido del archivo debajo de donde escribiste el nombre
     - Para grabar el archivo dale al botón verde en la parte superior. Dirá "Commit changes..." ó "Someter cambios..."
     - Presiona el botón verde en el diálogo que sale
