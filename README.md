# Ejercicios del Coding Bootcamp 2026

Los ejercicios se entregaran atravez de [GitHub.com](https://github.com) siguiendo los siguientes pasos. 
1. Crea una cuenta en [https://github.com/signup](https://github.com/signup)
2. Crea un repo con el nombre que desees (i.e.: bootcamp-2026)
3. Los ejercicios estaran en [https://github.com/scriptprojectsdev/bootcamp-2026](https://github.com/scriptprojectsdev/bootcamp-2026)
4. Cuando se emita un ejercicio en la clase lo subiras al repo que creaste
   - Para subir un archivo (lo mas recommendado) sigue estos pasos:
     - En la parte superior dale "Add file" &oacute; "A&ntilde;adir archivo"
     - Luego dale a "Upload file" &oacute; "Subir Archivo", asegurate que tu archivo tenga el mismo nombre que ejercicio
   - Para crear el archivo directamente en GitHub sigue estos pasos:
     - En la parte superior dale "Add file" &oacute; "A&ntilde;adir archivo"
     - Luego dale a "Create new file" &oacute; "Crear nuevo archivo"
     - Pon el nombre del archivo en la parte superior donde dice "Name your file..." &oacute; "Nombra tu archivo..."
     - Pega el contenido del archivo debajo de donde escribiste el nombre
     - Para grabar el archivo dale al boton verde en la parte superior. Dira "Commit changes..." &oacute; "Someter cambios..."
     - Presiona el boton verde en el dialogo que sale
