## Project Requirements

### 1. Navigation Bar (Required)
- The **navbar** should be **fixed at the top** and collapse into a **hamburger menu** on smaller screens.
- It should have links for **Dashboard, Users, and Settings.**

### 2. Dashboard Section (Required)
- Below the navbar, create a **dashboard overview** section with three **Bootstrap cards** displaying:
  - **Total Users**
  - **Active Users**
  - **Pending Approvals**
Each card should have an **icon**, a **title**, and a **number**.

### 3. User Registration Form (Required)
- Create a **form** where an admin can **add a new user**.
- The form should include:
  - **Full Name** input field
  - **Email Address** input field
  - **Role Selection** (Dropdown with options: Admin, User, Guest)
  - **Submit Button**
- Use **input validation** for required fields.
### 4. User Table (Required)
- Display a **Bootstrap-styled table** listing all registered users.
- The table should include:
  - **User ID**
  - **Name**
  - **Email**
  - **Role**
  - **Status** (Active or Inactive)
- Use **striped rows** and a **hover effect** to improve readability.
