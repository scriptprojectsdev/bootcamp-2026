# Personal Profile / Perfil Personal

Create a personal profile webpage introducing yourself.

**Requirements:**

- Display your name and a profile picture
- Write a short biography
- List your hobbies or interests
- Include links to social media or favorite websites
- Add contact information

**Concepts Learned:**

- HTML document structure
- Headings and paragraphs
- Lists
- Images
- Hyperlinks

#### Español

Crea una página web personal para presentarte.

**Requisitos:**

- Muestra tu nombre y una foto de perfil.
- Escribe una breve biografía.
- Enumera tus pasatiempos o intereses.
- Incluye enlaces a redes sociales o sitios web favoritos.
- Agrega información de contacto.

**Conceptos Aprendidos:**

- Encabezados y párrafos.
- Listas.
- Imágenes.
