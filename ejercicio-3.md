# Restaurant Menu / Menú de Restaurante

Create a restaurant menu webpage.

**Requirements:**

- Include the restaurant name and description
- Create sections for appetizers, entrees, drinks, and desserts
- Display item names and prices
- Add images for menu items (optional)
- Organize content using semantic HTML

**Concepts Learned:**

- Semantic HTML elements
- Lists and tables
- Page organization
- Content hierarchy

#### Español

Crea una página web para el menú de un restaurante.

**Requisitos:**

- Incluye el nombre y una descripción del restaurante.
- Crea secciones para entradas, platos principales, bebidas y postres.
- Muestra los nombres de los productos y sus precios.
- Agrega imágenes de los productos (opcional) [Unsplash](https://unsplash.com).
- Organiza el contenido utilizando HTML semántico.

**Conceptos Aprendidos:**

- Elementos semánticos de HTML.
- Listas y tablas.
- Organización de contenido.
- Jerarquía visual de la información.
