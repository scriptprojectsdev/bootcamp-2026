# Character Counter / Contador de Caracteres

Create a text area that displays a live character count as the user types.

**Requirements:**

- Text input area
- Live character count display
- Optional maximum character limit
- Warning message when approaching or exceeding the limit

**Concepts Learned:**

- Event listeners
- DOM manipulation
- User input handling
- Dynamic content updates

#### Español

Crea un área de texto que muestre en tiempo real la cantidad de caracteres escritos por el usuario.

**Requisitos:**

- Un área de texto para ingresar contenido.
- Un contador de caracteres en tiempo real.
- Un límite máximo de caracteres (opcional).
- Un mensaje de advertencia cuando se acerque o supere el límite.

**Conceptos Aprendidos:**

- Escuchadores de eventos (event listeners).
- Manipulación del DOM.
- Manejo de entradas del usuario.
- Actualización dinámica de contenido.
