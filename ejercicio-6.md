# Unit Converter / Conversor de Unidades

Create a converter that converts values between different units.

**Requirements:**

- User enters a numeric value
- Select source unit
- Select destination unit
- Display converted result

**Suggested Conversions:**

- Fahrenheit ↔ Celsius
- Miles ↔ Kilometers
- Pounds ↔ Kilograms

**Concepts Learned:**

- Conditional logic
- Mathematical operations
- Form handling
- Dynamic updates

#### Español

Crea una aplicación que convierta valores entre diferentes unidades de medida.

**Requisitos:**

- El usuario debe ingresar un valor numérico.
- Seleccionar la unidad de origen.
- Seleccionar la unidad de destino.
- Mostrar el resultado de la conversión.

**Conversiones Sugeridas:**

- Fahrenheit ↔ Celsius.
- Millas ↔ Kilómetros.
- Libras ↔ Kilogramos.

**Conceptos Aprendidos:**

- Lógica condicional.
- Operaciones matemáticas.
- Manejo de formularios.
- Actualización dinámica de contenido.
