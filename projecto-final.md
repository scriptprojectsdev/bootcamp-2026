### Weather App (Advanced) / Aplicación del Clima (Avanzada)

Create a weather application that retrieves real-time weather information from https://open-meteo.com and location data from https://www.latlng.work.

**Requirements:**

- Search by city or coordinates
- Retrieve weather data from Open-Meteo API
- Display current temperature and conditions
- Show additional weather details
- Responsive layout

**Concepts Demonstrated:**

- API integration
- Fetch API
- Async JavaScript
- JSON parsing
- Dynamic content rendering
- Error handling

#### Español

Crea una aplicación del clima que obtenga información meteorológica en tiempo real utilizando la API de Open-Meteo.

**Requisitos:**

- Buscar por ciudad o coordenadas.
- Obtener datos meteorológicos desde la API de Open-Meteo.
- Mostrar la temperatura actual y las condiciones climáticas.
- Mostrar información meteorológica adicional.
- Diseño adaptable para dispositivos móviles y de escritorio.

**Conceptos Demostrados:**

- Integración de APIs.
- Uso de Fetch API.
- JavaScript asíncrono.
- Procesamiento de datos JSON.
- Renderizado dinámico de contenido.
- Manejo de errores.

### AI ChatBot (Advanced) / ChatBot con Inteligencia Artificial (Avanzado)

Create a chatbot interface that communicates with an AI model through the OpenRouter API.

**Requirements:**

- Chat interface with message history
- User input field
- Send messages to OpenRouter API
- Display AI responses
- Loading indicator while waiting for responses

**Concepts Demonstrated:**

- API integration
- Async/await
- Fetch API
- JSON handling
- Dynamic UI updates

#### Español

Crea una interfaz de chatbot que se comunique con un modelo de inteligencia artificial mediante la API de OpenRouter.

**Requisitos:**

- Interfaz de chat con historial de mensajes.
- Campo de entrada para el usuario.
- Enviar mensajes a la API de OpenRouter.
- Mostrar las respuestas generadas por la IA.
- Indicador de carga mientras se espera la respuesta.

**Conceptos Demostrados:**

- Integración de APIs.
- Programación asíncrona con async/await.
- Uso de Fetch API.
- Manejo de JSON.
- Actualización dinámica de la interfaz.
